## Key Architecture Reminders
- `pipeline.py::analyze_ticker()` is the core entry point — intentionally monolithic (~270 lines)
- `scoring_config.py` is the single source of truth for all tunable constants
- `tools/calculator_tools.py` accepts `yf.Ticker` objects (not ticker strings)
- `compute_all_metrics()` returns the base metrics dict; pipeline.py adds `trailing_eps`, `book_value`, `fcf_per_share`, `current_price` on top of it before calling price targets
- Verdict boundaries: 80=Strong Buy, 65=Buy, 45=Hold, 30=Avoid, <30=Strong Avoid
